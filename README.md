### Event Dispatcher:


## Setup
```sh
$ git clone https://github.com/AslanAV/php-project-lvl2.git

$ make install
```


### Start Event Dispatcher
```shell
$ ./bin/ED
```


### Create New Event
```shell
$ ./bin/ED
```
```shell
$ e
```
